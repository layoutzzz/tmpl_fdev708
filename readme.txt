===========*Installation Template Instruction*==========
Please make following steps: 
1. Go to your FTP (root/templates) (where root - is main derectory for Boonex Dolphin script) 
2. Put 'tmpl_fdev' to 'root/templates' folder
3. Go to Admin Panel->Settings->Advanced Settings then choose "Deviation" skin as default one on your site.
4. Click "Clean All" in Admin Panel->Dashboard (to clean cache) 

Also you can see video how to do it here: http://www.layoutzzz.com/articles/how-to-install-dolphin-template.html
